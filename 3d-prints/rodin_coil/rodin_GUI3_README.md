# rodin_GUI3.py

Merged Tkinter/Matplotlib GUI for coil geometry + Biot-Savart visualization.

## Install

```bash
pip install numpy matplotlib
```

## Run

```bash
python rodin_GUI3.py
```

## Presets

- Solenoide
- Toroid
- Enkele winding
- Helmholtz
- Rodin-knot
- Rodin 3-phase CW/CCW
- Straight SawShape
- Curved SawShape
- Straight Starship
- Curved Starship

## Export

Use the buttons in **Visual / export**:

- `Save PNG`
- `Export coil CSV`

The CSV can be used with `coil_fourier_check.py` as a custom coil geometry.
