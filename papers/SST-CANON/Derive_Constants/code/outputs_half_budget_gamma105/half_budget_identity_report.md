# Half-budget identity gate

## Two-sector result

Assume the reduced finite-cell budget decomposes as

\[
E_{eff}=M_\phi+M_p.
\]

Let

\[
\gamma=\frac{M_\phi}{M_p}.
\]

Then

\[
M_\phi=\frac{\gamma}{1+\gamma}E_{eff}.
\]

For the self-dual value \(\gamma=1\),

\[
M_\phi=\frac{E_{eff}}{2}.
\]
