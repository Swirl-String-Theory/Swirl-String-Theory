## Immediate code-canon mismatch

The v0.8.12 main canon uses the corrected DDE mode condition

\[
\Omega=\omega_0-\kappa\sin(\Omega\tau).
\]

The available `SSTcore-v0.8.2.zip:src/delay_mode_selector.cpp` defines

```cpp
return Omega - omega0 - kappa * std::sin(Omega * tau);
```

which solves

\[
\Omega=\omega_0+\kappa\sin(\Omega\tau).
\]

So SSTcore should be patched before using it as the reproduction source for the v0.8.12 delay section.
