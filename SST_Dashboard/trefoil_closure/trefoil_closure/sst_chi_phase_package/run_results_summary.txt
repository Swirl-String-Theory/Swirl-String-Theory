SST chi-phase verification smoke test
=====================================

Environment note:
- pybind11 was not installed in this sandbox, so simulate_chi_phase.py used the included pure-Python fallback.
- On a local machine with pybind11 installed, the same script will auto-build sst_chi_phase.cpp.

Smoke-test results:
- max |c_chi/v_swirl - 1|        = 2.220e-16
- max |J_num/J_ana - 1|          = 7.813e-07
- omega_1/(omega_c), discrete    = 0.999999901954
- max |omega_n/(n omega_c)-1|    = 1.004e-04

Interpretation:
- The cancellation c_chi^2 = K_chi/I_chi = v_swirl^2 is numerically stable across the a_core/r_c sweep.
- The horn-loop spectrum L_chi = 2*pi*r_c reproduces omega_n ~= n*omega_c within the finite-difference tolerance.
- This is a Research Track sanity test only; it does not prove Q_chi -> Q_em, SU(2), SU(3), electroweak mixing, or the mass spectrum.
