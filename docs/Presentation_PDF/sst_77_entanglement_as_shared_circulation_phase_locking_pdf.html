SST Delayed Phase Dynamics

Interactive simulation of the minimal toy model for delay-coupled phase locking on shared circulation carriers, based on the paper by Omar Iskandarani.

INITIALIZING

System Parameters

20.0 s⁻¹

Base rotation speed of the carriers.

8.0 s⁻¹

Strength of phase-locking (depends on overlap O_AB).

0.35 s

Time for phase info to travel (L / ||v⟲||).

Governing Equations

dφ_A/dt = ω₀ + κ_{AB} sin(φ_B(t-τ) - φ_A(t))

dφ_B/dt = ω₀ + κ_{AB} sin(φ_A(t-τ) - φ_B(t))

Δ(t) = φ_A(t) - φ_B(t)

The system exhibits delayed Kuramoto-type phase locking. Depending on the dimensionless parameter 2κτ, the system will settle into an in-phase (Δ=0) or anti-phase branch.
